## How I use this

- **Cadence:** weekly, sent EOD Friday to stakeholder distribution
- **Delivery:** written in the team wiki (Confluence/Notion), link shared in the relevant exec Slack channel
- **Tone:** exec-first (TL;DR), data-backed (metrics section), action-oriented (decisions + next week)
- **Length:** 1 screen. If it's longer, I'm hiding problems.

## Why this format works

1. **Stakeholders can skim in 30 seconds** — TL;DR + health table tells the whole story
2. **Health status uses color + trend** — red-alone doesn't differentiate "was red, still red" from "was green, now red"
3. **"Decisions needed" is explicit** — status reports that end without asks become unread wallpaper
4. **Metrics are versus target + versus last week** — absolute numbers without context are noise
